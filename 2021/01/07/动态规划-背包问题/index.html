背包问题

问题陈述

​ 给你一个可装载重量为W的背包和N个物品，每个物品有重量和价值两个属性。其中第i个物品的重量为wt[i]，价值为val[i]，现在让你用这个背包装物品，最多能装的价值是多少？

问题解析

  1. 明确状态: 背包容量 和 物品数量
  2. 确定选择: 是否将物品装入背包
  3. 确定子问题: 放i-1个物品的最大价值，如果此时背包没装满，可以装最后一件物品，那么最大价值要再加上最后一件物品的价值，如果装满了那最大价值就是前i-1个物品的最大价值
  4. 定义状态转移方程: dp[m] [n] = Math.max(dp[m-i] [n],dp[m-1] [n-wt[i-1]] + val[i-1])。该转移方程的定义就是，前m件商品放在容量为n的背包中的最大价值是多少
  5. 明确初始条件，如果背包容量为0说明放满了，此时最大价值为0，如果没有物品了，自然最大价值也为0
  6. 处理边界情况：根据状态转移方程我们可以看出 n-wt[i-1] 必须要 >= 0,因为这样才可以装最后一件物品，如果小于0说明无法处理最后一件物品，那么该背包的最大价值就是dp[m-1] [n]

LeetCode 416 分割等和子集

题目定义

思路解析

  1. 题目要求问是否能将一个数组分割成两个等和子集，那么我们以将其简单化，抽象成背包问题，背包的容量就是一个子集的和(target = sum/2)，物品就是数组中的元素
  2. 确定状态: 很显然，本题就是数组中的元素与target
  3. 定义状态转移数组dp[i] [j].其定义为对于数组nums来说，前i个元素是否能组成和为j的子集
  4. 确定选择，即是否要将第i个元素放入前i-1个元素所组成的子集
  5. 写出状态转移方程 dp[i] [j] = dp[i-1] [j] (不放) || dp[i-1] [j-nums[i-1]] (放)
  6. 确定初始条件: 题目中明确是非空数组，因此当j为0时证明背包装满了，可以组成两个和为相等的子集，dp[0] [0] = false 因为nums[0] 是正整数凑不出0
  7. 扫清边界: 如果要放入最后一个元素，那么此时子集内就要预留出最后一个元素的位置，因此j-nums[j-1]要大于0

LeetCode 518 零钱兑换

题目定义

思路解析

  1. 题目要求用用不同面值的硬币凑成amount，很显然这也是个背包问题，背包容量就是amount，物品就是硬币的面值，和之前的背包问题不同，该题硬币是无限的，也就是说每个元素可以被重复使用
  2. 确定状态: amount 和 硬币面值
  3. 定义状态转移数组dp[i] [j].其定义为对于数组coins来说，使用前i个面值的硬币是否能凑测amount
  4. 确定选择，即是否要要用第i个面值的硬币
  5. 写出状态转移方程: dp[i] [j] = dp[i-1] [j] (不用最后一枚面值) + dp[i] [j-conis[i-1]] (用最后一枚面值);
  6. 如果需要用到最后一枚面值的硬币我们就需要考虑他的子问题什么，那就是还差一枚最后面值的硬币即可凑成amount
  7. 确定初始条件:题目说明了面值不能为0因此当amount为0时只有1种方式，即不用硬币
  8. 明确边界：如果我们要用到最后一枚硬币，那我们就需要预留出最后一枚面值硬币的位置因此j-conis[i-1]要大于0
